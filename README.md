# Python Lab Experiments

A comprehensive collection of Python experiments and practice problems organized by topic.

## 📁 Project Structure

This repository contains multiple Python experiment files organized in the `Python-lab-experiment-main/` directory.

### Experiment Categories

- **exp1.1 - exp1.8**: Foundational Python concepts and basic programs
- **exp2 (1-10)**: Data structures and algorithms
- **exp3 (2-7)**: Object-oriented programming and classes
- **exp4 (1-8)**: Functions, modules, and code organization
- **exp5 (1-4)**: File handling and I/O operations
- **exp7 (1-3)**: Exception handling and error management
- **exp8 (1-9)**: String manipulation and text processing
- **exp9 (1-3)**: List comprehensions and iterations
- **exp10 (1-4)**: Advanced data structures
- **exp11 (1-9)**: Additional practice problems and challenges
- **exp12 (1-5)**: Integration and combined concepts

### Special Files

- **ques1.py, ques2.py, ques3.py**: Question-based problems and solutions

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system
- A code editor or IDE (VS Code, PyCharm, etc.)

### Running Experiments

To run any experiment file:

```bash
python Python-lab-experiment-main/exp1.1.py
```

Or navigate to the directory and run individual files:

```bash
cd Python-lab-experiment-main
python exp1.1.py
```

## 📚 Content Overview

Each experiment file contains practical Python code examples and exercises covering:

- Variables, data types, and operators
- Control flow (if, loops)
- Functions and scope
- Data structures (lists, dictionaries, tuples, sets)
- Object-oriented programming
- File I/O and data handling
- String operations
- Exception handling
- Advanced programming concepts

## 💡 Usage Tips

1. **Sequential Learning**: Start with `exp1.1` and progress through the experiments
2. **Experimentation**: Modify and run files to understand different concepts
3. **Practice Problems**: Use `ques1.py`, `ques2.py`, `ques3.py` for practice after each section
4. **Version Control**: Track your progress with git commits

## 🔍 How to Navigate

- All experiment files are in the `Python-lab-experiment-main/` directory
- File names follow the pattern `exp[number] ([version]).py`
- Files within the same number group focus on similar concepts
